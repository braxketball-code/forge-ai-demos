# Quickstart: Real Estate AI Agent Automation

This package is tailored for the $1,500 Upwork real estate automation project.

## Core Components (from our Forge AI samples)
- lead-qualifier-agent.py: Scores leads 1-10, extracts details, suggests next action.
- tour-booking-agent.py: Handles scheduling and confirmation.
- listing-content.py: Generates descriptions, captions, emails.
- follow-up-nurturer.py: Personalized sequences for new leads/open houses.

## Setup
1. Clone this repo.
2. Install dependencies (Python 3, requests or your LLM client).
3. Configure your lead sources (webhook, email parser, or manual input).
4. Integrate calendar (Calendly API or Google Calendar).
5. Run qualifier on incoming leads.
